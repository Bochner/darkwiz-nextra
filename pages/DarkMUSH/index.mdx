# DarkMUSH Client

This section contains documentation for the DarkMUSH client features and windows, designed specifically for Dark Wizardry MUD.

## Available Windows

- [Affects Window](./affectswindow) - Displays active effects on your character
- [ASCII Map Window](./asciimapwindow) - Shows a visual map of your surroundings
- [Group Window](./groupwindow) - Displays information about your group members
